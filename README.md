# INF512-Unidad1-POO
Este repositorio fue creado como parte de la Unidad 1 de la materia INF512, con el fin de practicar los conceptos básicos de Programación Orientada a Objetos en C#.

## Descripción del proyecto
El proyecto contiene una clase llamada Carro, que representa un objeto real. Esta clase incluye:
- Atributos como la marca y la cantidad de gasolina.
- Un método que permite simular la acción de conducir un carro.
- 
## Objetivo
El objetivo del proyecto es demostrar el uso de:
- Clases
- Atributos
- Métodos
  
Utilizando así los principios básicos de la POO (Programación Orientada a Objetos).
